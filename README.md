# OpenRISC Presentation

I presented [OpenRISC](http://openrisc.github.io) architecture in [Tehran GNU/Linux Users Group](http://tehlug.org), and in this repository I shared files:

* `openrsic.odp` : The main presentation file
* `openrisc.pdf` : PDF presentation file
* `ALU.circ` : OpenRISC's class I ALU, implemented with `logisim`.
